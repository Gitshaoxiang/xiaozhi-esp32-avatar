# Xiaozhi Avatar Face
## 介绍

本项目基于虾哥开源的 ESP32 小智语音助手进行二次开发。硬件使用 M5Stack AtomS3R + Atomic Echo Base 搭配结构设计，修改显示表情包为 M5Stack Avatar Face。
